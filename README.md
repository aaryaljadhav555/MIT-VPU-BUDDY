# MIT VPU BUDDY 🤖

MIT VPU BUDDY is a modern AI-powered chatbot interface built using HTML, CSS, and JavaScript.  
It connects with an n8n webhook backend to provide intelligent campus assistance for MIT VPU students.

---

# ✨ Features

- Beautiful modern UI
- Responsive chat interface
- Typing animation
- AI chatbot integration
- n8n webhook support
- Session management
- Suggestion chips
- Smooth animations
- Dark futuristic theme
